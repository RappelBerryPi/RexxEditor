B - (Base) returns the number of complete days (not including the current day) since and incliuding th base date: 1 Jan 0001.
D - (Days) Returns the number of days, including the current day, so far in this year in the format 'ddd' (no leading zeros or blanks)
E - (European) returns the date in the format 'dd/mm/yy'
M - (Month) Returns the full English name of the current month, in mixed case.
N - (Normal) explicitly returns the date in default format.
O - (Ordered) Returns the date in the format 'yy/mm/dd' (suitible for sorting, etc.)
S - (Standard) Returns the date in the format 'yyyymmdd' (suitible for sorting, etc.) Note that this is one of the three forms recommended for ISO/R 2014-1971 (E).
W - (Weekday) Returns the English name for the day of the week, in mixed case.
